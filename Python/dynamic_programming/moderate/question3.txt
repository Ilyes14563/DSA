Edit Distance
Source: https://www.geeksforgeeks.org/edit-distance-dp-5/

Given two strings str1 and str2, find the minimum number of operations required to convert str1 to str2. Operations allowed: Insert, Remove, Replace a character.

Example 1:
Input: str1 = "sunday", str2 = "saturday"
Output: 3

Example 2:
Input: str1 = "cat", str2 = "cut"
Output: 1
