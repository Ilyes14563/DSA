Kth node from end of linked list
Source: https://www.geeksforgeeks.org/problems/nth-node-from-end-of-linked-list/1

Given a linked list consisting of N nodes, find the value of the kth node from the end of the linked list.

Example 1:
Input: LinkedList: 1->2->3->4->5, k = 2
Output: 4

Example 2:
Input: LinkedList: 1->2->3, k = 4
Output: -1
