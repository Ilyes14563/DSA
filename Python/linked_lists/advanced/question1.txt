Remove loop in Linked List
Source: https://www.geeksforgeeks.org/problems/remove-loop-in-linked-list/1

Given a linked list of N nodes. The task is to remove the loop in the linked list, if present.

Example 1:
Input: LinkedList: 1->3->4, and the last node (4) is connected to the second node (3).
Output: 1 3 4

Example 2:
Input: LinkedList: 1->8->3->4, and the last node (4) is connected to the first node (1).
Output: 1 8 3 4
