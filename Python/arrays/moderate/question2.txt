Count Inversions of an Array
Source: https://www.geeksforgeeks.org/count-inversions-array/

Given an array arr[], the task is to find the inversion count in the array. Two elements arr[i] and arr[j] form an inversion if arr[i] > arr[j] and i < j.

Example 1:
Input: arr[] = {1, 20, 6, 4, 5}
Output: 5

Example 2:
Input: arr[] = {2, 4, 1, 3, 5}
Output: 3
